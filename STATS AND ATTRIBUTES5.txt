These variables track the progression within an individual character.

*   :--- | :--- | :---

## III. Formulas for Interaction with Existing Mechanics

The existing game stats (Constitution, Resilience, Toughness, Stamina) will modify the pathogen's effects.

### A. Infection Check (Initial Exposure)

When a character is exposed to the pathogen, roll a check against their defense stats compared to the pathogen's ID50.

*   Infection_Chance = (Pathogen_ID50_Threshold / (Constitution + Resilience)) * Exposure_Severity

### B. Disease Progression (Damage & Fatigue)

*   **Body Part Damage:**
    Damage_per_Turn = (Base_Pathogen_Severity * Infection_Load_Multiplier) / Target_Toughness

*   **Fatigue Accumulation:**
    New_Fatigue_Rate = Base_Fatigue_Rate * (1 + (Pathogen_Severity_Multiplier / Character_Stamina))

### C. Recovery

The character's immune system passively fights the infection using their stats, similar to the Drift mechanics.

*   Recovery_Rate = (Constitution + Resilience) * Immunity_Level * Heal_Modifier

## Load Required to Get Ill (Infectious Dose)

*   Median Infectious Dose (ID50) Spearmanâ€“KÃ¤rber Method:
    log ID50 = log(highest dilution giving 100% CPE) + 0.5 - (total number of test units showing CPE / number of test units per dilution)
*   Multiplicity of Infection (MOI) (Cell Culture):
    MOI = Number of viral particles used per well / Number of cells originally seeded in the well
*   Plaque Assay Titer:
    Titer (PFU/ml) = Plaques counted * (1 / dilution) * volume correction

CPE(Cytopathic Effect) - The observable damage or morphological changes a virus causes to host cells in a culture.

## Multiplier During Illness (Viral/Bacterial Growth)

*   Viral Load (from PCR Ct value):
    viral RNA copies per mL = 133.3333 * e^((37.933 - Ct) / 1.418)
*   Bacterial Growth Rate (k) (generations/unit time):
    k = log10(Nt / N0) / (0.301 * t)
*   Bacterial Number (exponential growth):
    Nt = N0 * 2^n
    or
    Nt = N t = N0 * 10^(Kt)
    or
    N t = N0 * e^(Kt)
    (Where n is the number of generations, and K is the growth rate constant.)

## Load Value for Passing It On (Transmission Threshold)

*   Basic Reproduction Number (R0) (from transmission rate beta and removal rate gamma):
    R0 = beta / gamma
    (Alternatively: R0 = tau * c_bar * d (transmissibility tau, average contact rate c_bar, duration of infectiousness d))
*   Basic Reproduction Number (from exponential growth rate r and generation time distribution g(tau)):
    R0_hat = 1 / Integral[g(tau) * exp(-r*tau) d(tau)]
*   Viral Dynamics Model (R0):
    R0 = lambda * beta * k / (dT * dI * c)
    (lambda = target cell production rate, beta = infection rate, k = virus production rate, dT = target cell death, dI = infected cell death, c = virus clearance rate.)

# Formulas and Definitions (Reproduction Rate)

## Basic Reproduction Number (R0)

R0 is a theoretical, dimensionless number representing the average number of new infections caused by one infected person in a completely susceptible population with no interventions in place.

*   **Formula 1:**
    R0 = tau * c_bar * d
    (tau = transmissibility; c_bar = average rate of contact; d = duration of infectiousness)
*   **Formula 2 (from SIR model):**
    R0 = beta / gamma
    (beta = transmission rate; gamma = removal rate, or 1/d)

## Effective Reproduction Number (Rt or Re)

Rt measures the actual transmission potential of a disease at a specific time t in a  population, considering existing immunity levels, vaccination coverage, and ongoing control measures.

*   **Formula:**
    Rt = R0 * S/N
    (S = number of susceptible individuals; N = total population size)

### Interpretation

*   If R0 or Rt > 1: disease continues to spread and grow.
*   If R0 or Rt < 1: disease spread slows down and eventually dies out.
