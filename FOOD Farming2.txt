I. FOUNDATIONAL RULES
--------------------------------------------------------------------------------

1. Traits = Heritable (not locked to species)
2. Magnitude = Rolled per individual within inherited range
3. Species = Starting trait configuration, diverges through breeding
4. Physics/Environment = Modifies final expression (links to FormulasFull2)
5. Ploidy = Modifier on chromosome count, affects fertility/size
6. Diploidization = Long-term pressure toward lower chromosome counts

================================================================================

II. PLOIDY SYSTEM
--------------------------------------------------------------------------------

Ploidy = Number of chromosome sets

PloidyEnum:  
   - Haploid (1n)   = Reproductive cells only, non-viable as organism
   - Diploid (2n)   = Normal, baseline
   - Triploid (3n)  = Sterile (odd, can't divide evenly)
   - Tetraploid (4n)= Fertile with 4n, sterile with 2n
   - Pentaploid (5n)= Sterile
   - Hexaploid (6n) = Fertile with 6n, reduced with 3n
   - Higher         = Increasingly unstable

Ploidy Effects:  

Size Modifier: 
   - 2n:  1.0× base size
   - 3n:  1.3× base size
   - 4n: 1.5× base size
   - 6n: 1.8× base size

Fertility:  
   - Even ploidy (2n, 4n, 6n): Fertile with same ploidy
   - Odd ploidy (3n, 5n): Sterile, clone only
   - Cross-ploidy:  See compatibility matrix

Ploidy Compatibility Matrix:
   |     | 2n  | 3n  | 4n  | 5n  | 6n  |
   |-----|-----|-----|-----|-----|-----|
   | 2n  | 2n  | --  | 3n* | --  | 4n* |
   | 3n  | --  | --  | --  | --  | --  |
   | 4n  | 3n* | --  | 4n  | --  | 5n* |
   | 5n  | --  | --  | --  | --  | --  |
   | 6n  | 4n* | --  | 5n* | --  | 6n  |
   
   * = offspring ploidy, often sterile or reduced fertility
   -- = incompatible/no viable offspring

Ploidy Induction:
   - Heat stress:  5% chance per exposure
   - Cold stress: 3% chance per exposure
   - Chemical:  15% chance (colchicine equivalent)
   - Magic: Variable by spell
   - Natural mutation: 0.1% per generation

================================================================================

III. DIPLOIDIZATION PRESSURE
--------------------------------------------------------------------------------

Long-term Selection Favors Fewer Chromosomes

Stable Species Threshold: n ≤ 25 pairs (diploid)

Diploidization Rate: 
   - Per generation: 0.1% chance of chromosome fusion
   - Accelerated by:  Inbreeding, environmental stability
   - Resisted by: Outbreeding, environmental variability

High Chromosome Count Risks:
   - n > 50: +10% mutation rate (harmful)
   - n > 75: +20% developmental failure
   - n > 100: +30% sterility chance

Farming Implication:
   - Super-yield polyploids require cloning to maintain
   - Natural breeding drifts toward stable (lower) counts
   - "Game-breaking" cultivars need active management

================================================================================

IV. DIVERGENCE SYSTEM (Universal)
--------------------------------------------------------------------------------

Divergence Score = Σ(Trait Differences) / Total Traits

Trait Difference Scoring:
   - Same Enum: 0
   - Adjacent Enum: 1
   - Different Enum: 2
   - Opposite Enum: 3

Magnitude Difference: 
   - |Range_A - Range_B| × Weight

Chromosome Difference:
   - |Count_A - Count_B| / Max(Count_A, Count_B) × 0.5

Total Divergence = (Enum Diff + Magnitude Diff + Chromosome Diff) / Max Possible

--------------------------------------------------------------------------------

Divergence Thresholds:  

0. 00 - 0.15: Same Species (Variant)
0.16 - 0.35:  Subspecies
0.36 - 0.55: Related Species (Cross-fertile)
0.56 - 0.75: Distant Species (Hybrid possible, fertility risk)
0.76 - 1.00: Separate Species (Incompatible)

--------------------------------------------------------------------------------

New Species Establishment:  

Requirements:
   1. Divergence Score ≥ 0.36 from origin
   2. Stable traits for 3 generations
   3. Population threshold (breeding pool exists)

On Establishment:
   - New species_id generated
   - Origin recorded (parent species)
   - Trait configuration becomes new baseline
   - Magnitude ranges reset to current averages

================================================================================

V.  STERILITY TYPES
--------------------------------------------------------------------------------

Type A: F1 Sterile (Permanent)
   - Hybrid offspring always sterile
   - Propagation only via cloning
   - Occurs when:  Divergence 0.65 - 0.75 OR odd ploidy
   - Example:  Mule, Seedless banana

Type B: F1 Sterile, F2+ Fertile (Recoverable)
   - First generation sterile
   - Backcross to parent species restores fertility
   - F2 from backcross = fertile but trait dilution
   - Occurs when: Divergence 0.56 - 0.64

Type C:  Reduced Fertility (Degrading)
   - Each hybrid generation = reduced fertility chance
   - Fertility % = 100 - (Generation × Divergence × 50)
   - Eventually stabilizes or goes sterile
   - Occurs when:  Divergence 0.45 - 0.55

================================================================================

VI. COMPATIBILITY CHECK (Universal)
--------------------------------------------------------------------------------

Function CheckCompatibility(Entity_A, Entity_B):

   # Kingdom check
   IF Entity_A.kingdom != Entity_B.kingdom:
      RETURN Incompatible
   
   # Class check
   IF Entity_A. class != Entity_B.class:
      RETURN Incompatible
   
   # Subclass check (allows adjacent)
   IF Entity_A.subclass != Entity_B.subclass:
      IF NOT Adjacent(Entity_A. subclass, Entity_B.subclass):
         RETURN Incompatible
   
   # Ploidy check
   ploidy_compatible = CheckPloidyMatrix(Entity_A.ploidy, Entity_B.ploidy)
   IF NOT ploidy_compatible:
      RETURN Incompatible
   
   # Divergence check
   divergence = CalculateDivergence(Entity_A, Entity_B)
   
   IF divergence ≤ 0.35:
      RETURN Fertile
   
   IF divergence ≤ 0.55:
      RETURN ReducedFertility (Type C)
   
   IF divergence ≤ 0.64:
      RETURN F1Sterile_Recoverable (Type B)
   
   IF divergence ≤ 0.75:
      RETURN F1Sterile_Permanent (Type A)
   
   IF divergence > 0.75:
      RETURN Incompatible

================================================================================

VII. MUTATION SYSTEM
--------------------------------------------------------------------------------

Mutation Chance:  Per gene, per breeding cycle
   - Base:  0.01 (1%)
   - High chromosome count (n>50): +0.005
   - Inbreeding: +0.02
   - Environmental stress: +0.01 to +0.05
   - Magic exposure: Variable

Mutation Outcomes:  

Beneficial (25% base):
   - Expand magnitude range upward
   - New trait combination
   - Increased resistance
   - Modified by:  Cultivation/Husbandry skill

Neutral (50%):
   - Color variation
   - Minor sensory shift
   - No mechanical effect

Harmful (25% base):
   - Reduce magnitude range
   - Increased decay/disease susceptibility
   - New vulnerability
   - Developmental defects

Polyploidy Mutation (Rare):
   - 0.1% base chance
   - Doubles chromosome set
   - Size increase, fertility change

Chromosome Fusion (Diploidization):
   - 0.1% base chance in stable populations
   - Reduces chromosome count by 1 pair
   - No immediate phenotype change

II. PLANT CROSS-CLASS COMPATIBILITY
--------------------------------------------------------------------------------

Same Class:  Full recombination
Adjacent Class: Partial recombination (shared pairs only)
Distant Class:  Incompatible

Adjacency: 
   Flowering ↔ Conifer (distant seed plants)
   Fern ↔ Moss (spore-bearing)
   Algae ↔ None (isolated)

                         PART IV: REPRODUCTION METHODS
██████████████████████████████████████████████████████��█████████████████████████

================================================================================

I. SEXUAL REPRODUCTION (Genetic Recombination)
--------------------------------------------------------------------------------

Applies to:  All organisms with compatible mates

Process:
   1. Compatibility check (class, subclass, ploidy, divergence)
   2. Gamete formation (meiosis - halves chromosome count)
   3. Fertilization (combines gametes)
   4. Offspring genome generated
   5. Mutation check per gene
   6. Phenotype expression calculated

Outcome:
   - Genetic variation
   - Trait recombination
   - Magnitude range inheritance
   - Possible new trait combinations

Plant-Specific (Pollination):
   - Pollen lands on compatible stigma
   - Pollen tube grows to ovule
   - Fertilization occurs
   - Seed forms with mixed genome

Success Factors:
   - Genetic compatibility
   - Timing (both parents fertile)
   - Environmental conditions
   - Pollinator presence (if required)

Failure Modes:
   - Pollen tube doesn't grow
   - Self-incompatibility
   - Chromosome mismatch
   - Environmental stress

--------------------------------------------------------------------------------

II. ASEXUAL REPRODUCTION (Cloning)
--------------------------------------------------------------------------------

Methods:  

A. Cutting (Plants)
   - Success Rate: Species-dependent (0.3 - 0.9)
   - Requirements: Moisture, rooting hormone, warmth
   - Result: Exact clone

B. Tuber/Bulb (Plants)
   - Success Rate: High (0.8 - 0.95)
   - Requirements: Dormancy period, proper storage
   - Result: Exact clone

C. Layering (Plants)
   - Success Rate: Higher than cutting (0.6 - 0.95)
   - Requirements: Parent plant support, soil contact
   - Result: Exact clone

D. Fragmentation (Fungi, some Plants)
   - Success Rate: Variable (0.2 - 0.8)
   - Requirements: Viable fragment size
   - Result: Exact clone

E. Budding (Fungi, some Animals)
   - Success Rate: High (0.7 - 0.9)
   - Requirements: Mature parent
   - Result: Exact clone

Cloning Effects:
   - No genetic variation (except rare mutation)
   - Preserves all traits exactly
   - Maintains ploidy level
   - Required for sterile organisms (odd ploidy)
   - Mutation rate:  0.001 (1/10th of sexual)

--------------------------------------------------------------------------------

III. GRAFTING (Plants Only)
--------------------------------------------------------------------------------

Process: 
   1. Select rootstock (determines root traits)
   2. Select scion (determines above-ground traits)
   3. Compatibility check (genus-level minimum)
   4. Vascular alignment (cambium layers must touch)
   5. Fusion check (success/fail)

Outcome:
   - Chimera:  Two genomes in one plant
   - Rootstock traits:  Environment tolerance, disease resistance
   - Scion traits: Fruit, flavor, yield
   - No genetic mixing

Success Factors:
   - Genetic distance:  Same species (0.9), Same genus (0.6), Different genus (0.2)
   - Alignment quality: Skill-based
   - Environment: Humidity, temperature
   - Clean cuts, no infection

Failure Modes:
   - Genetic distance too large
   - Poor alignment
   - Infection
   - Tissue desiccation

--------------------------------------------------------------------------------

IV. HYBRIDIZATION (Cross-Species Sexual)
--------------------------------------------------------------------------------

Process:
   1. Parent A (Species 1) × Parent B (Species 2)
   2. Enhanced compatibility check
   3. Chromosome pairing attempt
   4. Viability check
   5. Sterility determination

Outcomes by Divergence:
   - 0. 36-0.45: Fertile hybrid, new species potential
   - 0.46-0.55: Reduced fertility (Type C)
   - 0.56-0.64: F1 sterile, backcross recoverable (Type B)
   - 0.65-0.75: F1 sterile permanent (Type A)

Hybrid Vigor:
   - First generation often shows enhanced traits
   - +10-20% to yield, size, resistance
   - Degrades in subsequent generations unless stabilized

--------------------------------------------------------------------------------

V. POLYPLOIDY INDUCTION
--------------------------------------------------------------------------------

Triggers:
   - Heat shock:  5% per exposure
   - Cold shock: 3% per exposure
   - Chemical (colchicine equivalent): 15% per application
   - Magic: Variable
   - Natural mutation: 0.1% per generation

Outcomes:
   - Chromosome doubling
   - Size increase (1. 3-1.8×)
   - Fertility change (odd = sterile, even = fertile with same ploidy)
   - Trait amplification

Stabilization:
   - Polyploid hybrids may become compatible
   - Requires generations to stabilize
   - Success = new stable species

================================================================================

I. INHERITANCE TYPES
--------------------------------------------------------------------------------

A.  MENDELIAN (Simple Dominant/Recessive)

Applies to: Binary traits (present/absent)

Notation:
   - DD = Homozygous Dominant → Trait expressed (strong)
   - Dr = Heterozygous → Trait expressed (moderate)
   - rr = Homozygous Recessive → Trait not expressed

Traits using Mendelian:  
   - Disease Resistance (Resistant > Susceptible)
   - Poison Presence (Poison > Neutral)
   - Natural Weapons (Present > Absent)

--------------------------------------------------------------------------------

B. CODOMINANCE (Both Express)

Applies to: Enum traits that can combine

Notation:
   - AA = Pure A
   - AB = Both A and B expressed
   - BB = Pure B

Traits using Codominance:  
   - Flavor Type (Sweet × Sour = Sweet-Sour)
   - Season Tolerance (Summer × Spring = Spring-Summer)
   - Texture Type (Soft × Crunchy = Soft-Crunchy)
   - Color (Red × White = Red-White spotted)

--------------------------------------------------------------------------------

C. INCOMPLETE DOMINANCE (Blend)

Applies to: Continuous/range traits

Notation:
   - AA = Full A expression
   - AB = Intermediate (blend of A and B)
   - BB = Full B expression

Traits using Incomplete Dominance:
   - Flavor Intensity (Strong × Weak = Medium)
   - Texture Intensity (Strong × Weak = Medium)
   - Potency Level (High × Low = Medium)
   - Growth Time (Fast × Slow = Medium)
   - Size (Large × Small = Medium)
   - All magnitude ranges

--------------------------------------------------------------------------------

D. POLYGENIC (Multiple Genes → One Trait)

Applies to: Complex traits

Multiple gene pairs contribute to single phenotype.  
More dominant alleles = stronger expression.

Traits using Polygenic:
   - Macro Ratios (Protein/Carb/Fat distribution)
   - Yield (multiple modifiers)
   - Intelligence/Trainability
   - Overall Size/Mass

================================================================================

II. MAGNITUDE RANGE INHERITANCE
--------------------------------------------------------------------------------

For traits with [min, max] ranges:  

Offspring. min = random_choice(Parent_A.min, Parent_B.min)
Offspring.max = random_choice(Parent_A.max, Parent_B.max)

Range Shift via Mutation:
   - Beneficial: Expand range by 5-10%
   - Harmful: Contract range by 5-10%
   - Direction: Random or environmental pressure

Selective Breeding Effect:
   - Consistent selection shifts range over generations
   - 3+ generations same direction = stable shift

================================================================================

III. TRAIT EXPRESSION FORMULA
--------------------------------------------------------------------------------

Final_Value = Roll_Within_Range × Environment_Modifier × Ploidy_Modifier × Skill_Modifier

Where:
   - Roll_Within_Range:  random(trait. min, trait.max)
   - Environment_Modifier: 0.5 - 1.5 based on condition match
   - Ploidy_Modifier: 1.0 (2n), 1.3 (3n), 1.5 (4n), 1.8 (6n)
   - Skill_Modifier: 1.0 + (skill_level × 0.005)

   # Kingdom check
   IF Parent_A.kingdom != Parent_B.kingdom:
      RETURN BreedingResult(success=false, reason="Kingdom mismatch")
   
   # Class check
   IF Parent_A.class != Parent_B.class:
      RETURN BreedingResult(success=false, reason="Class mismatch")
   
   # Subclass check (allows adjacent)
   IF Parent_A.subclass != Parent_B.subclass:
      IF NOT Adjacent(Parent_A.subclass, Parent_B.subclass):
         RETURN BreedingResult(success=false, reason="Subclass incompatible")
   
   # Ploidy check
   offspring_ploidy = CheckPloidyMatrix(Parent_A.ploidy, Parent_B.ploidy)
   IF offspring_ploidy == null:
      RETURN BreedingResult(success=false, reason="Ploidy incompatible")
   
   # Divergence calculation
   divergence = CalculateDivergence(Parent_A, Parent_B)
   IF divergence > 0.75:
      RETURN BreedingResult(success=false, reason="Divergence too high")
   
   # Determine fertility type
   fertility = DetermineFertility(divergence, offspring_ploidy)
   
   # Fertility roll (for reduced fertility types)
   IF fertility == TypeC:
      fertility_chance = 100 - (divergence × 100)
      IF random(100) > fertility_chance:
         RETURN BreedingResult(success=false, reason="Fertility roll failed")
   
   # ═══════════════════════════════════════════════════════
   # PHASE 2: GENOME GENERATION
   # ═══════════════════════════════════════════════════════
   
   offspring = new OrganismGenome()
   offspring.origin_species_id = Parent_A.species_id
   offspring. generation = max(Parent_A.generation, Parent_B.generation) + 1
   offspring. ploidy = offspring_

   offspring. ploidy = offspring_ploidy
   offspring.chromosome_pairs = CalculateChromosomePairs(Parent_A, Parent_B, offspring_ploidy)
   
   mutations = []
   
   # ═══════════════════════════════════════════════════════
   # PHASE 3: TRAIT INHERITANCE
   # ═══════════════════════════════════════════════════════
   
   FOR each trait in traits:
      inheritance_type = GetInheritanceType(trait)
      
      # Allele selection
      allele_from_A = random_choice(Parent_A.trait. allele_a, Parent_A.trait.allele_b)
      allele_from_B = random_choice(Parent_B.trait.allele_a, Parent_B.trait.allele_b)
      
      # Apply inheritance rules
      SWITCH inheritance_type:
         CASE Mendelian:
            offspring.trait. value = ResolveDominance(allele_from_A, allele_from_B)
         
         CASE Codominant:
            IF allele_from_A != allele_from_B: 
               offspring.trait.value = Combine(allele_from_A, allele_from_B)
            ELSE:
               offspring. trait.value = allele_from_A
         
         CASE IncompleteDominance:
            offspring.trait.range = Average(Parent_A.trait.range, Parent_B.trait.range)
         
         CASE Polygenic: 
            offspring.trait = CalculatePolygenic(Parent_A, Parent_B, trait)
      
      # Store alleles
      offspring.trait.allele_a = allele_from_A
      offspring.trait.allele_b = allele_from_B
      
      # Mutation check
      mutation_chance = GetMutationChance(offspring)
      IF random() < mutation_chance:
         mutation = ApplyMutation(offspring.trait)
         mutations.append(mutation)
   
   # ═══════════════════════════════════════════════════════
   # PHASE 4: MAGNITUDE RANGE INHERITANCE
   # ═══════════════════════════════════════════════════════
   
   FOR each range_trait in traits:
      offspring.trait.range. min = random_choice(Parent_A.trait.range.min, Parent_B.trait.range.min)
      offspring.trait.range.max = random_choice(Parent_A.trait.range.max, Parent_B.trait.range.max)
      
      # Range mutation
      IF random() < mutation_chance:
         offspring.trait.range = ShiftRange(offspring.trait.range)
         mutations.append(range_mutation)
   
   # ═══════════════════════════════════════════════════════
   # PHASE 5: SPECIAL CHECKS
   # ═══════════════════════════════════════════════════════
   
   # Polyploidy induction (rare)
   IF random() < 0.001:
      offspring.ploidy = IncreasePloidy(offspring.ploidy)
      offspring.fertility = UpdateFertilityForPloidy(offspring. ploidy)
   
   # Chromosome fusion (diploidization)
   IF offspring.stable_generations > 5 AND random() < 0.001:
      offspring.chromosome_pairs -= 1
   
   # Hybrid vigor check
   hybrid_vigor = (divergence >= 0.2 AND divergence <= 0.4)
   IF hybrid_vigor: 
      ApplyHybridVigor(offspring)  # +10-20% to key traits
   
   # ═══════════════════════════════════════════════════════
   # PHASE 6: DIVERGENCE UPDATE
   # ═══════════════════════════════════════════════════════
   
   offspring.divergence_from_origin = CalculateDivergence(offspring, offspring. origin_species_id)
   
   # Check stability
   prev_divergence = Parent_A.divergence_from_origin
   IF abs(offspring.divergence_from_origin - prev_divergence) < 0.05:
      offspring.stable_generations = Parent_A.stable_generations + 1
   ELSE:
      offspring.stable_generations = 0
   
   # New species check
   new_species = false
   IF offspring.stable_generations >= 3 AND offspring.divergence_from_origin >= 0.36:
      new_species = EstablishNewSpecies(offspring)
   
   # ═══════════════════════════════════════════════════════
   # PHASE 7: RETURN RESULT
   # ═══════════════════════════════════════════════════════
   
   RETURN BreedingResult(
      success = true,
      offspring_genome = offspring,
      fertility_type = fertility,
      hybrid_vigor = hybrid_vigor,
      mutations = mutations,
      ploidy_change = (offspring.ploidy != 2n),
      new_species = new_species
   )

Function GetMutationChance(organism):
   
   base_chance = 0.01  # 1%
   
   # High chromosome count increases mutation
   IF organism.chromosome_pairs > 50:
      base_chance += 0.005
   IF organism.chromosome_pairs > 75:
      base_chance += 0.005
   IF organism.chromosome_pairs > 100:
      base_chance += 0.01
   
   # Inbreeding increases mutation
   IF organism. parent_a_id == organism.parent_b_id:
      base_chance += 0.02
   
   # Environmental stress (passed as context)
   base_chance += context.stress_modifier  # 0.01 to 0.05
   
   # Magic exposure (passed as context)
   base_chance += context.magic_modifier  # Variable
   
   RETURN base_chance

Function ApplyMutation(trait):
   
   outcome_roll = random(100)
   
   # Determine outcome type
   IF outcome_roll < 25:
      outcome = Beneficial
   ELIF outcome_roll < 75:
      outcome = Neutral
   ELSE:
      outcome = Harmful
   
   # Modify by skill
   outcome = ModifyBySkill(outcome, context.skill_level)
   
   # Apply mutation based on trait type
   IF trait.has_range:
      IF outcome == Beneficial:
         trait. range = ExpandRange(trait.range, 0.05, 0.10)
      ELIF outcome == Harmful:
         trait.range = ContractRange(trait.range, 0.05, 0.10)
      # Neutral = no change
   
   IF trait.has_enum:
      IF outcome == Beneficial OR outcome == Neutral:
         trait.value = ShiftEnum(trait.value, random_direction)
      ELIF outcome == Harmful: 
         trait.value = ShiftEnumNegative(trait.value)
   
   RETURN {
      trait: trait. name,
      old_value: original_value,
      new_value: trait.value,
      outcome: outcome
   }

Function CalculateDivergence(Entity_A, Entity_B):
   
   total_difference = 0
   max_possible = 0
   
   # Enum trait differences
   FOR each enum_trait in traits:
      diff = GetEnumDistance(Entity_A.trait. value, Entity_B.trait.value)
      # 0 = same, 1 = adjacent, 2 = different, 3 = opposite
      total_difference += diff
      max_possible += 3
   
   # Magnitude range differences
   FOR each range_trait in traits:
      range_a = Entity_A.trait.range
      range_b = Entity_B.trait.range
      
      # Normalize to 0-1 scale
      diff = abs((range_a.min + range_a.max)/2 - (range_b. min + range_b.max)/2)
      normalized_diff = diff / max_trait_value
      
      total_difference += normalized_diff × weight
      max_possible += weight
   
   # Chromosome difference (weighted at 0.5)
   chrom_diff = abs(Entity_A.chromosome_pairs - Entity_B.chromosome_pairs)
   chrom_max = max(Entity_A.chromosome_pairs, Entity_B.chromosome_pairs)
   normalized_chrom = (chrom_diff / chrom_max) × 0.5
   
   total_difference += normalized_chrom
   max_possible += 0.5
   
   RETURN total_difference / max_possible

Function DetermineFertility(divergence, ploidy):
   
   # Odd ploidy = always sterile (Type A)
   IF ploidy IN [3n, 5n, 7n]: 
      RETURN TypeA_Permanent
   
   # Divergence-based fertility
   IF divergence <= 0.35:
      RETURN Fertile
   
   IF divergence <= 0.55:
      RETURN TypeC_Reduced
   
   IF divergence <= 0.64:
      RETURN TypeB_Recoverable
   
   IF divergence <= 0.75:
      RETURN TypeA_Permanent
   
   RETURN Incompatible

Function EstablishNewSpecies(organism):
   
   # Generate new species ID
   new_species_id = GenerateSpeciesID()
   
   # Record origin
   species_record = {
      species_id: new_species_id,
      origin_species_id: organism.origin_species_id,
      established_generation: organism.generation,
      founding_genome: organism.genome. copy(),
      trait_baseline: organism.traits.copy(),
      magnitude_ranges: organism.ranges.copy()
   }
   
   # Update organism
   organism.species_id = new_species_id
   organism.origin_species_id = new_species_id  # Self-referential now
   organism.divergence_from_origin = 0. 0
   organism.stable_generations = 0
   
   # Register in species database
   RegisterNewSpecies(species_record)
   
   RETURN true

Function ApplyHybridVigor(organism):
   
   # Boost key traits by 10-20%
   vigor_bonus = random(0.10, 0.20)
   
   # Apply to yield
   organism.traits.yield. range.min *= (1 + vigor_bonus)
   organism.traits.yield.range.max *= (1 + vigor_bonus)
   
   # Apply to size
   organism.traits.size. range.min *= (1 + vigor_bonus)
   organism.traits.size.range.max *= (1 + vigor_bonus)
   
   # Apply to disease resistance
   organism.traits.disease_resistance.range.min += vigor_bonus × 0.5
   organism. traits.disease_resistance.range. max += vigor_bonus × 0.5
   
   # Cap at 1.0 for ratio traits
   organism.traits.disease_resistance.range.max = min(1.0, organism.traits.disease_resistance.range.max)
